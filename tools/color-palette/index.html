カラーパレット生成ツールの使い方

      
スペースキーを押すか「新しいパレット」ボタンをクリックするだけで、調和の取れた5色の配色を自動生成します。気に入った色は🔒ボタンでロックすれば、他の色だけが変更されます。

      
デザインの配色で迷ったら

      
    
        
  • 60-30-10ルール — メインカラー60%、サブカラー30%、アクセントカラー10%の比率で使うとバランスが良い
    • 
        
  • 類似色 — 色相環で隣り合う色を組み合わせると落ち着いた印象に
    • 
        
  • 補色 — 正反対の色を組み合わせるとダイナミックな印象に
    • 
        
  • モノトーン — 同じ色相の明度・彩度違いで統一感を出す
    • 
      

      
WebデザインでのカラーコードI

      
Webデザインでは主にHEXコード（#FF5733）やRGB（rgb(255, 87, 51)）が使われます。このツールでは両方のコードを表示し、ワンクリックでコピーできます。CSS変数として一括エクスポートも可能です。
